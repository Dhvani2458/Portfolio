# 💼 Dhvani Tandel - Personal Portfolio

Welcome to my personal portfolio website! This portfolio showcases my technical skills, projects, internships, certifications, and achievements as a developer. It has been designed to provide recruiters, collaborators, and visitors with an overview of my work, experience, and interests in software development and cybersecurity.

## 🌐 Live Demo

🔗 **Portfolio:** *https://dhvani-tandel.netlify.app*

## 📖 About

This portfolio is a modern, fully responsive website built using HTML, CSS, and JavaScript. It highlights my journey as a Computer Science student and developer, featuring projects, internship experiences, technical expertise, certifications, and a contact form for professional communication.

The website focuses on clean UI/UX design, smooth animations, responsive layouts, and interactive user experience.
